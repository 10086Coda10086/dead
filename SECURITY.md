# Security Policy

## Supported Version / 支持版本

一般情况下，我们只维护 **最新版本** *(最新的 `commit`, `tag` 和 `release`)*，所以请确保更新到最新版本来避免可能的漏洞和攻击.

## Reporting / 提交报告

如果你：
- 发现了严重影响应用**安全性或性能**的漏洞
- 发现了可能导致本项目**受到损害**的内容
- 发现了可能威胁本项目**协作者的人身安全**的内容
- 其他**危险内容**
